Customer Purchase Analysis Dashboard Project

Project Overview

The Customer Purchase Analysis Dashboard project focuses on analyzing customer purchase behavior to identify key revenue drivers and business opportunities. The objective is to transform raw customer data into meaningful insights and present them through clear and interactive dashboards.

Business Objective

Understand customer purchasing patterns
Identify top performing product categories
Analyze regional sales performance
Evaluate the impact of promotions on revenue
Identify high value customers

Tools & Technologies Used

Excel – Data cleaning, pivot tables, and dashboard creation
SQL (MySQL) – Data analysis using queries
Power BI – Data visualization and interactive dashboard design

Dataset

Customer Purchase Dataset
Contains customer demographics, purchase amount, product category, region, loyalty status, and promotion usage

Project Workflow

Data Preparation (Excel)

Converted raw data into structured table format
Cleaned and validated data
Created calculated fields (e.g., Promotion Status Yes/No)
Built pivot tables for category, region, and customer analysis

Data Analysis (SQL)

Created database and customer_data table
Inserted sample records
Performed analysis using:
SELECT statements
GROUP BY
Aggregate functions (SUM)

Analyzed:

Sales by Category
Sales by Region
Loyalty Status performance
Promotion impact
Top customers by purchase amount

Data Visualization (Excel & Power BI)

Built visuals to analyze:
Sales by Product Category
Sales by Region
Sales by Loyalty Status
Sales by Promotion Usage
Sales by Satisfaction Score
Top 10 Customers by Sales
Designed a structured dashboard layout
Added slicers for interactive filtering

Key Analysis Performed

Category-wise revenue comparison
Region-wise sales performance
Loyalty segment analysis
Promotion effectiveness analysis
Customer satisfaction impact on spending
Identification of high-value customers

Key Insights

Identified top revenue-generating product categories
Found regions contributing the highest sales
Observed that loyalty status influences purchase behavior
Evaluated the effect of promotions on total revenue
Recognized top customers contributing significant revenue

This project demonstrates the ability to analyze customer behavior, generate business insights, and present results through dashboards using Excel, SQL, and Power BI.
